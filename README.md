# Tank Battle - 坦克大战

一个使用 Python 和 Pygame 开发的经典坦克大战游戏。

## 游戏特色

- 精美的像素风格画面
- 经典的坦克大战操作体验
- 多种敌人类型（普通、快速、重型）
- 丰富的地形效果（砖墙、钢墙、水域、草地）
- 爆炸和烟雾特效
- 多个关卡挑战

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 射击 |
| ESC / P | 暂停游戏 |

## 安装和运行

### 从源码运行

1. 确保已安装 Python 3.8+
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行游戏：
   ```bash
   python main.py
   ```

### 下载可执行文件

从 [Releases](https://github.com/ganecheng/tank_battle_python/releases) 页面下载最新版本的可执行文件。

## 游戏目标

- 保护基地（鹰）不被摧毁
- 消灭所有敌方坦克
- 成功通过所有关卡

## 敌人类型

- **普通坦克** (红色)：标准速度和火力
- **快速坦克** (粉色)：移动速度快，但装甲薄弱
- **重型坦克** (深红)：装甲厚，需要多次击中才能摧毁

## 地形说明

- **砖墙** (棕色)：可以被子弹摧毁
- **钢墙** (灰色)：无法被子弹摧毁
- **水域** (蓝色)：坦克无法通过，子弹可以飞过
- **草地** (绿色)：坦克可以隐藏其中
